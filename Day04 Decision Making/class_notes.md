# Day 2 - User Input & Data Types 
## What is Input()?

`input()` is used to take information from the user while the program is running 

syntax:

```python
variable = input("Rajesh")
print(name)
```

### Important Rules 
`input()` always returns a string (`str`), even if the user enters numbers.

Example:

Input
25

Output Type:
<class 'str'>
